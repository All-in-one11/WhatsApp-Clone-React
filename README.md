## General info
This project is a WhatsApp Web clone. Users can create public chat rooms.
	
## Technologies
Project is created with:
* React
* React Context API
* Material-Ui v4.11.0
* React Router v5.2.0
* React Player v2.6.2
* Firebase: Authentication, Cloud Firestore(db), Storage, Hosting
